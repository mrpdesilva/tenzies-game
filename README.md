# 🎲 Tenzies Game

A fun and interactive **dice game** built using **React.js**, where the goal is to roll all dice until every die shows the same number.

---

## 📌 About the Game

**Tenzies** is a simple yet addictive game:

- Roll the dice until all values are the same  
- Click on dice to **freeze (hold)** their values  
- Continue rolling until all dice match  
- Try to win in the **least number of rolls**  

---

## ⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/mrpdesilva/tenzies-game.git
cd tenzies-game

Install dependencies:

npm install

Run the development server:

npm start

---

## 🧪 Build for Production

npm run build
